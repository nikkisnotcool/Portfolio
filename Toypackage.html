"My First Blocks" Package Redesign

The original package Janod used for their box series was functional, but lacked the imagery of a nurturing children’s toy. This design seeks to combine this with increased functionality. The double window provides another real visual in addition to the icons on the sides of the box displaying all 6 designs. The hand done type is friendly, homemade, inviting, and creates a more nurturing persona for the package. The pattern also does this, inspiring images of baby blankets. The features of this toy are condensed into more easily digestible forms of communication that allows a scanner to consume a lot of information quickly.
